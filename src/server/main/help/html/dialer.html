Dialer

The Dialer is used to enter phone numbers to place a call.

To place a call: enter the number by pressing keypad keys and then press .

Lookup: If there are matching names in Contacts or numbers in Call History they are displayed in the order of Contacts and Call History Item.

For example, type 2 and 3 to find "Adam" in Contacts. To search Call History, type at least three digits to start the lookup.

Use / to select a number before pressing .

Press Options for options:

  • Send Message: opens the SMS composer to send a message to the number.
  • Save to Contacts: saves the number to the Contacts application.
